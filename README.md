# بنانا اتوماسیون پروژه‌ها

این مخزن شامل تمام فایل‌های افزونه در همین سطح است تا بتوانید مستقیماً کل پوشه را زیپ و در وردپرس نصب کنید (بدون پوشه‌ی تودرتو).

- فایل اصلی افزونه: `benana-automation-projects.php`
- پوشه‌های دارایی‌ها، کلاس‌ها، زبان و قالب‌ها همگی در همین مسیر قرار دارند.
- بروزرسانی خودکار از `update.json` مخزن انجام می‌شود و با دکمه «بررسی دستی بروزرسانی» در تنظیمات می‌توانید وضعیت را فوراً چک کنید.

نسخه کنونی: 1.3.3
